PERCEPTRONIUM
=============

Perceptronium is a lightweight toolkit for building Behler--Parrinello
symmetry-function descriptors and training neural-network interatomic
potentials. It combines a fast C++ descriptor-generation core with Python
utilities for data conversion, packaging, and model training, making it easy
to move from atomistic trajectories and energies to trainable ML-ready inputs.
